# Rostex Font Files

This directory contains the Rostex font files used for the header branding.

To use custom Rostex fonts:
1. Download Rostex-Regular.woff2 and Rostex-Bold.woff2
2. Place them in this directory
3. The layout.tsx file already references these fonts

If you don't have the font files, you can:
- Download from Google Fonts: https://fonts.google.com
- Or use a commercial font foundry
- Or replace with an alternative font in app/layout.tsx
